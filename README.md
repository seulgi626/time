# 1만 시간의 법칙

## 배포링크
URL : https://seulgi626.github.io/time/

1. 시멘틱 마크업
- header: 페이지의 제목과 주요 로고 영역
- section: 콘텐츠의 주제별 그룹
- form: 사용자 입력을 받는 폼 영역
- footer: 저작권 정보 및 하단 로고

2. css 변수 (:root) 활용
색상과 폰트 등 자주 사용되는 값을 :root 가상 클래스에 변수로 정의하여 관리했습니다. 이를 통해 디자인의 일관성을 유지하고, 추후 유지보수를 용이하게 만들었습니다.
